# Hotel Booking App

Дипломный проект: сайт-агрегатор для поиска, просмотра и бронирования гостиниц.

## Описание проекта

Приложение для бронирования гостиниц с регистрацией, входом, просмотром гостиниц/номеров, бронированием, поддержкой и админ-панелью.

## Текущий статус

Проект полностью реализован. Фронтенд на React с TypeScript и Vite включает все необходимые модули: аутентификацию (регистрация и вход), просмотр и поиск гостиниц и номеров, бронирование номеров с проверкой доступности, управление гостиницами и номерами (для администраторов), управление пользователями, техническую поддержку и админ-панель.

Бэкенд на NestJS с MongoDB полностью реализован и интегрирован с фронтендом. Включает модули аутентификации с JWT, пользователей, гостиниц и номеров (CRUD операции), бронирований с проверкой доступности, технической поддержки (запросы от клиентов и управление сотрудниками).

## Технологии

- **Frontend**: React, TypeScript, Vite, React Router
- **Backend**: NestJS, MongoDB, Mongoose, JWT, Passport
- **Дополнительно**: Node.js, npm, MongoDB Community Server ([установка](https://docs.mongodb.com/manual/installation/))

## Установка и запуск

### Требования
- Node.js >= 18
- npm >= 9
- MongoDB (локально на порту 27017)

### Шаги
1. Клонируйте репозиторий: `git clone <URL>`
2. Установите зависимости: `npm install`
3. Запустите MongoDB.
4. Запустите приложение: `npm start` (запускает бэкенд на порту 3000 и фронтенд на порту 5173 одновременно)

Альтернативно, можно запустить отдельно:
- Бэкенд: `cd server && npm install && npm run start:dev` (порт 3000)
- Фронтенд: `npm run dev` (порт 5173)
