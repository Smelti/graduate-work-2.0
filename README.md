# Hotel Booking App

Дипломный проект: сайт-агрегатор для поиска, просмотра и бронирования гостиниц.

## Описание проекта

Приложение для бронирования гостиниц с регистрацией, входом, просмотром гостиниц/номеров, бронированием, поддержкой и админ-панелью.

## Текущий статус

Фронтенд полностью реализован на React с TypeScript и Vite. Включает модули аутентификации (регистрация и вход), просмотра списка гостиниц и пользователей, управления гостиницами (добавление, редактирование, удаление) и детальной информации о гостиницах.

Бэкенд на NestJS с MongoDB находится в активной разработке. Интеграция фронтенда и бэкенда завершена для модулей пользователей, аутентификации, списка гостиниц и операций CRUD с гостиницами. Функционал бронирования реализован на бэкенде, но еще не интегрирован с фронтендом. Модуль технической поддержки пока не реализован и находится в разработке.

## Технологии

- **Frontend**: React, TypeScript, Vite, React Router
- **Backend**: NestJS, MongoDB, Mongoose, JWT, Passport
- **Дополнительно**: Node.js, npm, MongoDB Community Server ([установка](https://docs.mongodb.com/manual/installation/))

## Установка и запуск

### Требования
- Node.js >= 18
- npm >= 9
- MongoDB (локально на порту 27017)

### Шаги
1. Клонируйте репозиторий: `git clone <URL>`
2. Установите зависимости: `npm install`
3. Запустите MongoDB.
4. Запустите бэкенд: `cd server && npm install && npm run start:dev` (порт 3000)
5. Запустите фронтенд: `npm run dev` (порт 5173)
